# Syntax Cheat Sheet

 This will help you remember syntax in coding.

 ## Special Characters


### Parenthesis (Parens)
```js
()
```

### Curly Braces
```js
{

}
```

### Angle Brackets
```html
< > <!--opening tag-->
</ > <!--closing tag-->
```

### Semicolon
```js
;
```

### Colon
```js
:
```

### Double Quotes
```js
" "
```

### Single Quotes
```js
' '
```

### Backticks
```js
` `
```

### Template Literal String
```js
`This is some text that ${myVariable1} will be   ${myVariable2} taken literally. `
```
